## 심사 기준
  [현장 발표 평가]
  - 팀별로 심사위원들의 부스 방문 시 발표하여 심사

  [발표 평가 점수]
  - 총점 20점 만점으로 평가
  - 주제의 독창성 (6점)
  - 목표의 타당성 (4점)
  - 결과의 완성도 (6점)
  - 설계의 활용도 (4점)

  [최종 심사결과]
